* What is it?
~po4a-docker~ is standalone [[https://github.com/mquinson/po4a][po4a]].

* Dependency
Depend Docker.
#+begin_src bash
  brew install docker
  brew cask install docker

  # docker start-up test
  docker version
#+end_src

* How to use?
Run ~po4a-docker~ like as below.

#+begin_src bash
  $ docker container run -i --rm conao3/po4a:0.55 --version
